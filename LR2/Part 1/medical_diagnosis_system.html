Потоки данных

1. Ввод данных: Пользователь вводит симптомы (температура, кашель, насморк и т.д.)

2. Обработка: Машина вывода анализирует симптомы через правила из базы знаний

3. Вывод: Система определяет наиболее вероятный диагноз с фактором уверенности

4. Объяснение: Система показывает, какие правила были активированы

5. Рекомендации: Система выдает план лечения и рекомендации

Атрибуты основных классов

Заболевание:

  • название: String
  • код_МКБ: String
  • инкубационный_период: Integer (дни)
  • заразность: Boolean
  • степень_тяжести: Enum (легкая, средняя, тяжелая)

Симптом:

  • название: String
  • интенсивность: Integer (1-10)
  • длительность: Integer (дни)
  • частота_проявления: Float (0-1)

Пациент:

  • возраст: Integer
  • наличие_хронических_заболеваний: Boolean
  • температура_тела: Float
  • иммунный_статус: Enum (слабый, нормальный, хороший)
База правил (12 правил)
R001 CF: 0.85
Определение гриппа типа A
IF (Условия):
температура > 38.5°C AND резкое_начало = TRUE AND ломота_в_мышцах = высокая AND головная_боль = сильная AND слабость = выраженная
THEN (Заключение):
диагноз = "Грипп типа A"
💊 Рекомендации: Постельный режим, противовирусные препараты (осельтамивир), обильное питье, жаропонижающие при температуре >38.5°C
R002 CF: 0.90
Определение обычной ОРВИ
IF (Условия):
температура = 37.5-38°C AND насморк = TRUE AND боль_в_горле = умеренная AND кашель = легкий AND постепенное_начало = TRUE
THEN (Заключение):
диагноз = "ОРВИ"
💊 Рекомендации: Симптоматическое лечение, покой, теплое питье, промывание носа солевыми растворами
R003 CF: 0.75
Выявление COVID-19
IF (Условия):
потеря_обоняния = TRUE OR потеря_вкуса = TRUE AND температура > 37.5°C AND (сухой_кашель = TRUE OR одышка = TRUE)
THEN (Заключение):
диагноз = "Подозрение на COVID-19"
💊 Рекомендации: Срочно сдать ПЦР-тест, самоизоляция, мониторинг сатурации кислорода, при одышке - обращение к врачу
R004 CF: 0.80
Определение бактериальной ангины
IF (Условия):
боль_в_горле = очень_сильная AND налет_на_миндалинах = TRUE AND увеличенные_лимфоузлы = TRUE AND температура > 38°C AND НЕТ насморка AND НЕТ кашля
THEN (Заключение):
диагноз = "Ангина (бактериальная)"
💊 Рекомендации: Консультация врача, антибиотики (амоксициллин), полоскание горла антисептиками, жаропонижающие
R005 CF: 0.70
⚠️ Выявление осложнения - пневмонии
IF (Условия):
длительность_заболевания > 7_дней AND температура > 38°C AND кашель_с_мокротой = TRUE AND боль_в_груди = TRUE AND одышка = TRUE
THEN (Заключение):
диагноз = "Риск пневмонии"
🚨 Рекомендации: СРОЧНО обратиться к врачу, рентген грудной клетки, возможна госпитализация, антибиотикотерапия
R006 CF: 0.95
Неосложненная простуда
IF (Условия):
температура < 37.5°C AND легкий_насморк = TRUE AND легкое_недомогание = TRUE AND НЕТ сильной_головной_боли
THEN (Заключение):
диагноз = "Легкая простуда"
💊 Рекомендации: Домашнее лечение, витамин C (1000 мг/день), теплое питье, полноценный отдых
R007 CF: 0.78
Определение острого бронхита
IF (Условия):
продуктивный_кашель = TRUE AND хрипы_в_легких = TRUE AND длительность_кашля > 5_дней AND температура = 37-38°C
THEN (Заключение):
диагноз = "Острый бронхит"
💊 Рекомендации: Муколитики (АЦЦ, Амброксол), отхаркивающие средства, обильное теплое питье, при бактериальной природе - антибиотики
R008 CF: 0.88
⚠️ Высокий риск осложнений у пожилых
IF (Условия):
возраст > 65_лет AND (диагноз = "Грипп" OR диагноз = "ОРВИ") AND хронические_заболевания = TRUE
THEN (Заключение):
риск_осложнений = "Высокий"
⚠️ Рекомендации: Обязательная консультация врача, ежедневный мониторинг состояния, рассмотреть противовирусную терапию, контроль хронических заболеваний
R009 CF: 0.92
⚠️ Особенности течения у детей до 3 лет
IF (Условия):
возраст < 3_лет AND температура > 38.5°C AND отказ_от_еды = TRUE
THEN (Заключение):
действие = "Требуется осмотр педиатра"
⚠️ Рекомендации: Вызов педиатра на дом, контроль температуры каждые 2 часа, жаропонижающие по возрасту (парацетамол, ибупрофен), обильное питье
R010 CF: 0.82
Выявление синусита как осложнения
IF (Условия):
насморк > 10_дней AND боль_в_области_пазух = TRUE AND гнойные_выделения = TRUE AND головная_боль_усиливается_при_наклоне = TRUE
THEN (Заключение):
диагноз = "Синусит"
💊 Рекомендации: Консультация ЛОР-врача, антибиотики (амоксициллин+клавулановая кислота), сосудосуживающие капли (не более 5 дней), промывание пазух
R011 CF: 0.75
Показания к антибиотикотерапии
IF (Условия):
(диагноз = "Ангина" OR диагноз = "Синусит" OR диагноз = "Бронхит") AND бактериальная_природа = вероятна AND длительность > 5_дней
THEN (Заключение):
лечение = "Антибиотики показаны"
💊 Рекомендации: Назначение антибиотиков ТОЛЬКО врачом (амоксициллин, азитромицин), полный курс 5-7 дней, пробиотики для защиты микрофлоры
R012 CF: 1.0
🚨 КРИТИЧЕСКИЕ СИМПТОМЫ
IF (Условия):
(одышка = тяжелая OR сатурация_O2 < 92%) OR (температура > 40°C AND не_снижается_жаропонижающими) OR спутанность_сознания = TRUE
THEN (Заключение):
действие = "СРОЧНАЯ госпитализация"
🚨 Рекомендации: Немедленный вызов скорой помощи, транспортировка в стационар, интенсивная терапия
Алгоритм работы системы
1
Сбор симптомов пациента
Пользователь вводит данные о симптомах через интерфейс: температура тела, наличие кашля, насморка, болей в горле, длительность симптомов, характер начала заболевания и другие проявления.
2
Формирование рабочей памяти
Система создает список фактов на основе введенных данных и сохраняет их в рабочей памяти. Например: [температура=39.0, кашель=сухой, насморк=нет, начало=резкое, слабость=сильная]
3
Активация правил
Машина вывода проверяет условия всех правил в базе знаний. Правила, условия которых выполняются, добавляются в список активированных правил с их факторами уверенности.
4
Разрешение конфликтов
Если активировано несколько правил с разными диагнозами, система выбирает наиболее подходящее на основе: 1) Максимального фактора уверенности, 2) Количества совпавших условий, 3) Специфичности диагноза.
5
Вывод заключения
Система формирует диагноз с указанием уверенности (например: "Грипп типа A (85% уверенность)") и определяет список рекомендаций из соответствующего правила.
6
Генерация объяснений
Модуль объяснений создает понятное обоснование диагноза, показывая: какие симптомы учитывались, какие правила сработали, почему выбран именно этот диагноз, а не альтернативные.
7
Формирование рекомендаций
Модуль рекомендаций создает план действий: необходимые медикаменты, режим лечения, когда обращаться к врачу, признаки ухудшения состояния, которые требуют немедленной медицинской помощи.
8
Вывод результата
Пользователю отображается полный результат диагностики: установленный диагноз, уровень уверенности, обоснование, рекомендации по лечению, предупреждения об опасных симптомах.
