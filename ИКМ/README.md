# Прогнозирование спроса на велопрокат

Проект по дисциплине "Интеллектуальные системы" (ПГУ).

## Задача

Регрессия: по погодным условиям, времени суток и дню недели предсказать количество велопрокатов за час.

## Датасет

**UCI Bike Sharing Dataset** — данные системы Capital Bikeshare (Вашингтон, 2011-2012).

- Источник: [UCI ML Repository](https://archive.ics.uci.edu/dataset/275/bike+sharing+dataset)
- Файл: `bike_data/hour.csv` — 17,379 записей
- Целевая переменная: `cnt` (количество прокатов за час)
- Признаки: сезон, месяц, час, день недели, погода, температура, влажность, ветер

## Структура проекта

```
├── bike_data/          # датасет (hour.csv, day.csv)
├── models/             # обученная модель (best_model.pkl)
├── tests/              # тесты (pytest)
│   └── test_basic.py
├── train.py            # обучение и оценка моделей
├── predict.py          # модуль предсказания
├── app.py              # веб-интерфейс (Gradio)
├── requirements.txt    # зависимости
└── README.md
```

## Установка и запуск

```bash
pip install -r requirements.txt

# Обучение модели
python train.py

# Запуск веб-интерфейса
python app.py

# Запуск тестов
pytest tests/ -v
```

## Результаты

| Модель | MAE | RMSE | R2 |
|--------|-----|------|----|
| RandomForest | **32.67** | 51.48 | **0.9163** |
| GradientBoosting | 34.86 | 51.92 | 0.9149 |

Лучшая модель: **RandomForest** — объясняет 91.6% дисперсии спроса, средняя ошибка ~33 проката/час.
